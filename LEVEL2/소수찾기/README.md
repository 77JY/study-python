숫자로된 문자열이 매개변수로 주어질 때 맨 앞자리가 0 이 아닌 모든 숫자의 조합중 소수인 수가 몇개인지 찾는문제

1.먼저 숫자에 있는 수를 itertools를 사용해서 나올수 있는 모든 경우의수를 딕셔너리로 나타낸다 (겹치는 숫자가 있어도 굳이 넣고 제거하면 낭비같아서 딕셔너리안에 해당하는 숫자가 있는지 검사하고 없으면 넣는다) 2.모든 경우의수가 되는 숫자를 받아와서 그 숫자가 소수인지 검사하고 소수면 answer + 1 마지막에 리턴

문제를 중간에 풀긴했지만 효율적이지 못해서 다시 풀었다
모든 숫자의 조합을 미리 리스트로 만든뒤 에라토스테네스의 체를 사용해서 그걸 들고와 비교했는데
최고 4초까지 걸려서 그냥 딕셔너리 안에 있는 숫자를 하나하나 검사하게 바꾸니 오히려 시간이 낮아졌는데 아마 딕셔너리의 수가 많지않아서 그런갑다 ..
