# 내가 문제를 푼 방식
처음에 생각한 풀이방식

1.숫자 최소값을 구한다 
2.최소값의 개수를 구함 
3.최소값의 수만큼 총 길이에서 빼고 + 1해줌 [ 하나는 있어야하니 ] 
4.그리고 nums[1]에서 -1한 값에서 나눠줌  
5.나머지가 1을 넘을경우 + 1 6.출력


# 풀면서 생긴 문제
2묶음씩 묶을 경우 남은 길이/1를 해버려서 그냥 숫자 그대로 출력되어버린다
ex: 10억 = 10억회

# 해결방법
묶음이 2개일 경우 결과값에서 -1하면 완성
